update_one& mongocxx::model::update_one::upsert ( bool  upsert)

Sets the upsert option.

When upsert is false, if no document matches the filter, the update operation does nothing. However, by specifying upsert as true, this operation either updates a matching document or inserts a new document using the update specification if no matching document exists. By default upsert is false.

Parameters
upsertIf set to true, creates a new document when no document matches the query criteria. The server side default is false, which does not insert a new document if a match is not found.
